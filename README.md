# One-Dimensional P-N Junction Diode Model

This project simulates a one-dimensional abrupt silicon p-n junction diode using Python.

## Features

- Equilibrium charge density, electric field, and electrostatic potential
- Equilibrium and bias-dependent energy-band diagrams
- Ideal Shockley current-voltage characteristic
- Shockley-Read-Hall recombination-generation current
- Series resistance
- Reverse breakdown
- Metal-semiconductor Schottky contact

## Requirements

- Python 3
- NumPy
- Matplotlib
